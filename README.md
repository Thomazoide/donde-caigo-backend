# Donde Caigo - Backend

DondeCaigo APP es una aplicación que permitirá a los usuarios que la usen, buscar fiestas/after/carretes de forma segura, con verificacion de identidad y confirmacion
del coordinador de la fiesta y el solicitante de ingreso, contará con chat y publicaciones, asi como geolocalización para detectar las más cercanas.

En desarrollo temprano.
